What's up Bootstrap

Bootstrap is the most popular CSS Framework for developing responsive and mobile-first websites.
with jsDelivr, a free open source CDN, and a template starter page, you can make really cool things in your website.

Open Source and Mobile First

Bootstrap is a free and open source framework that uses the mobile-first technology for front-end development Bootstrap contains CSS and Javascript based on design templates for different types of content like:

  • Typography
  • Forms
  • Buttons
  • Navigation
    • etc...
Bootstrap includes a responsive grid system that scales up to 12 columns as the device or viewport size increases.

Some of Bootstrap Components

Accordion

Build vertically collapsing accordions in combination with the Collapse JavaScript plugin.

This is the first item's accordion body. It is shown by default, until the collapse plugin adds the appropriate classes that we use to style each element. These classes control the overall appearance, as well as the showing and hiding via CSS transitions. You can modify any of this with custom CSS or overriding our default variables. It's also worth noting that just about any HTML can go within the .accordion-body, though the transition does limit overflow.
Buttons

Use Bootstrap’s custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

Progress Bar
